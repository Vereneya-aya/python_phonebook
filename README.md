# python_phonebook

## Файловая система
*Функции:*
1. Добавление данных
2. Удаление данных
3. Изменение данных
4. Вывод данных
5. Копирование данных 
6. Выход из программы

**ui.py** - отвечает за взаимодействие программы с пользователем

**main.py** - главный файл, который вызывает всю цепочку

**db** - папка, в которой хранится 2 файла-справочника
